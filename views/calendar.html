Some notes about the table design

The text within the table cells and circles should align center.

To make the one digit numbers align right with the two digit numbers, before each one digit number that is not within a circle there is a numsp-html-entity.

Only if a one digit number appears with a circle around it, it will not have the numeric space, but will aligned centered within the circle.

To fill the half background of a cell that shows a start or end circle a background-gradient was used with intermediate stops at 50%

Important: in this example there is a month that has more weeks than the other months. In Flex all months have the same hight if they are next to each other. This was not made part of these specks, but should be done during implentation.

Purpose of this specs was to come up with a reasonable definition of the spacing within the table of individual months.
